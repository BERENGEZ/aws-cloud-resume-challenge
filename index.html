Infrastructure Architecture

A completely self-hosted, bare-metal cloud environment. Here is how traffic flows from a user's browser to the local database.

🌐

1. Cloudflare Tunnel

Bypasses ISP CGNAT, providing secure inbound routing and SSL termination without exposing open router ports to the public internet.

🖥️

2. Proxmox & Ubuntu

A bare-metal hypervisor running a dedicated Ubuntu Server VM, intelligently allocating isolated CPU and RAM resources.

🚦

3. Nginx Web Server

Acts as a reverse proxy. It serves the static frontend (HTML/CSS/JS) and securely routes /api/ requests to the backend.

⚙️

4. Python Flask API

Managed by Gunicorn and systemd. This custom REST API processes POST requests and executes the core backend logic.

🗄️

5. SQLite Database

Locally stores visitor telemetry and metrics. It is completely isolated from the web, accessible only by the internal API.

🚀

6. CI/CD Pipeline

GitHub Actions triggers a self-hosted runner directly on the Ubuntu VM to automatically pull and deploy new code upon every commit.
